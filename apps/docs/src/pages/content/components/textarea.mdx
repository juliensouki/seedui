## Overview

<OverviewDemo example={doc.example} />

Textarea is a multi-line text field for longer-form content such as messages, descriptions, comments, or notes. It shares the same label and styling conventions as Input for a consistent form experience. The isResizable prop controls whether users can drag to resize the field. Use Textarea when the expected input is more than a single line — for short values like names or emails, use Input instead.
