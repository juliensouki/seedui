## Overview

<OverviewDemo example={doc.example} />

Toggle is a switch control for binary settings that take effect immediately, such as enabling notifications, dark mode, or feature flags. Unlike a checkbox — which typically requires a form submission — a toggle visually communicates that the change applies right away. It supports an optional label that appears next to the switch for context. Use the size prop to fit the toggle into dense toolbars or more spacious settings pages.

## Usage

### Basic toggle

A controlled switch with a label. The change takes effect immediately, making it ideal for settings that don't require a form submission.

<ComponentPlayground code={doc.usageExamples[0].code} />

### Sizes

Three sizes are available. Small works in dense settings panels, while large provides a bigger hit target for touch interfaces.

<ComponentPlayground code={doc.usageExamples[1].code} />

### Disabled

A disabled toggle is visually muted and cannot be interacted with. Use it when a setting is locked or unavailable.
