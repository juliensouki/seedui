## Overview

<OverviewDemo example={doc.example} />

Popover displays rich content in a floating panel anchored to a trigger element. Use it for contextual menus, secondary actions, or inline details that don't warrant a full modal. Unlike Tooltip, which shows only a short text label, Popover can contain any React content — buttons, forms, or lists. It supports configurable alignment and automatic positioning to stay within the viewport. For simple text hints on hover, use Tooltip instead.
