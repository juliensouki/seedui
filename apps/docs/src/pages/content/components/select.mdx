## Overview

<OverviewDemo example={doc.example} />

Select is a dropdown component for choosing a single value from a predefined list of options. It includes built-in search filtering, making it suitable for both short lists and longer option sets. Use it for form fields where the user must pick from a known set of values — such as country selectors, category pickers, or role assignments. Each option can include an optional icon for richer visual context. For free-form text entry, use Input instead.
