## Overview

<OverviewDemo example={doc.example} />

Divider creates a visual separation between content sections. Use it to break up long pages, separate list items, or distinguish groups of related controls. It supports both horizontal and vertical orientations — horizontal dividers work well between stacked sections, while vertical dividers are useful inside flex layouts like toolbars or side-by-side panels. The spacing prop lets you control how much breathing room surrounds the line.
