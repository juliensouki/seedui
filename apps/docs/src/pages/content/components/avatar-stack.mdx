## Overview

<OverviewDemo example={doc.example} />

AvatarStack renders a set of Avatar components in an overlapping row. Use the max prop to limit the number of visible avatars — any overflow is shown as a "+N" badge. The size prop is applied uniformly across all avatars in the stack. This component is ideal for showing team members, collaborators, or participants in a space-efficient way.

## Usage

### Basic stack

Avatars overlap naturally. Each avatar receives a border to maintain visual separation.

<ComponentPlayground code={doc.usageExamples[0].code} />

### With overflow

Set max to limit visible avatars. A "+N" indicator shows how many are hidden.

<ComponentPlayground code={doc.usageExamples[1].code} />

### Sizes

The size prop controls all avatars in the stack uniformly.

<ComponentPlayground code={doc.usageExamples[2].code} />

### Direction

Controls which avatar appears on top. "left" (default) puts the first avatar on top, "right" puts the last on top.
