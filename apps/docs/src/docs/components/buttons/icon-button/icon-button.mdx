<section id="section-usage">
## Usage

### Variants

Like Button, IconButton supports filled and transparent variants. Use transparent for toolbar actions or subtle controls.

<ComponentPlayground code={doc.usageExamples[0].code} scope={iconScope} />

### Color schemes

Apply a color to match the action context. Primary for main actions, neutral for general-purpose, and error for destructive ones.

<ComponentPlayground code={doc.usageExamples[1].code} scope={iconScope} />

### Sizes

Small works well in dense toolbars, medium is the default, and large is suited for prominent standalone actions.

<ComponentPlayground code={doc.usageExamples[2].code} scope={iconScope} />

### Disabled

A disabled IconButton is visually muted and non-interactive. Use it when the action is temporarily unavailable.

<ComponentPlayground code={doc.usageExamples[3].code} scope={iconScope} />

### Custom color

Pass any hex color to the color prop and SeedUI will automatically generate a full shade scale for hover, focus, and active states. Works with both filled and transparent variants.

<ComponentPlayground code={doc.usageExamples[4].code} scope={iconScope} />
