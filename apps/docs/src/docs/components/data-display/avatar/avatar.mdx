<section id="section-usage">
## Usage

### With image

Pass a src prop to display an image. The name prop provides alt text and serves as a fallback if the image fails to load.

<ComponentPlayground code={doc.usageExamples[0].code} layout={doc.usageExamples[0].layout} />

### With initials

When no image is provided, initials are generated from the name prop. Single names show one letter, multi-word names show first and last initials.

<ComponentPlayground code={doc.usageExamples[1].code} layout={doc.usageExamples[1].layout} />

### Sizes

Use sm for dense layouts, md for general use, and lg for profile pages or headers.

<ComponentPlayground code={doc.usageExamples[2].code} layout={doc.usageExamples[2].layout} />

### Colors

Use color to differentiate avatars by role, team, or status when no image is available.

<ComponentPlayground code={doc.usageExamples[3].code} layout={doc.usageExamples[3].layout} />

### Fallback icon

When neither src nor name is provided, a generic person icon is displayed.

<ComponentPlayground code={doc.usageExamples[4].code} layout={doc.usageExamples[4].layout} />
